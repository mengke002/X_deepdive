# X_deepdive

A social network data mining and analysis toolkit for discovering insights from X (Twitter) data.

## Features

- **Network Analysis**: Build social graphs, calculate PageRank/Betweenness centrality, community detection
- **Behavioral Analytics**: Activity patterns, content efficiency metrics, user fingerprinting
- **LLM-Powered Insights**: Deep qualitative analysis using AI models

## Quick Start

```bash
pip install -r requirements.txt
python -m src.deep_mining
```
